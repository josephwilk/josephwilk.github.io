Errors are Inevitable - 2024

Materials: ChatGPT4o, LLMs & OpenAI Text to voice.

Themes: AI.

A simple prompt instruction for a Large Language Model (LLM) that asks for all permutations of 2 letters for a certain length.

List ALL the strings with a length of seven characters and only contains characters "a" and "b".
Do not miss a single string.

This prompt can be used to demonstrate LLMs will always error. Expanding the length of the generated string, will ALWAYS eventually result in an error in the answer. This is an innate part of LLMs current design and something that cannot be fixed. We can always find problems where the LLM cannot map every possible computable answer and these simple examples can be expanded to real world knowledge. LLMs operate through memory not intelligence and hence struggle to adapt to something unseen. We avoid the word "hallucinate" to describe this error as it's a miscategorisation of a system that does not work.

Voice reading is provided using OpenAIs text to voice service.

Based on research from: Hallucination is Inevitable: An Innate Limitation of Large Language Models
